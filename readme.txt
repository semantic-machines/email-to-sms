Утилита предназначена для рассылки sms-сообщений с электронного почтового ящика.
Сообщения должны быть представлены в следующем формате: тема сообщения - номера телефонов в виде 
+71231231212 или 71231231212 разделённые точкой с запятой, тело сообщения - текст для смс.

Конфигурирационный файл программы ./conf/options.js
